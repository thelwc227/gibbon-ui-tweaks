# gibbon-ui-tweaks
A Tampermonkey script written using Copilot AI to customize the look of certain elements on gibbonedu.

GreasyFork One-Click install: https://greasyfork.org/en/scripts/562588-gibbon-ui-tweaks/

Pleae install Tampermonkey before use

Chrome: https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en

Firefox: https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/

Open the tampermoneky GUI and press 'Create a new script', then paste in the gibbon-ui-tweaks.js into the code editor and press save in the file tab.

This project is created written Copilot AI and a member of the ICHK community which has nothing else to do with their free time. It is in no way affiliated with gibbonedu and please use responsibly and at your own risk

Known bugs:

-Better timetable causes tutor time text to dissapear

-ATL entries and activity selections do not function if the script is turned on, please turn off this script to acess theese features on the gibbon website.
